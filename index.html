  • IdentityServer 是一个框架，一个使用 OpenID Connect 和 OAuth2 协议为现代 Web 应用和 API 实现单点登录和访问控制的可托管组件。 它能够支持很多种类的客户端，比如移动设备，Web ，SPA 和桌面应用且能集成到新的或现有的架构中。
  • 在这些页面中，你可以找到关于 IdentityServer 以及来自我们和社区的相关项目的更新，文档和信息。
